OnLib

Сайт онлайн-библиотека

Проект «OnLib» направлен на разработку и запуск полнофункционального веб-ресурса для чтения книг в онлайн-режиме. Сайт будет отличаться структурированным каталогом, адаптивным дизайном и простой навигацией, что обеспечит пользователям максимальное удобство при работе с цифровым контентом.

Основные возможности

Аутентификация пользователей: Полный цикл регистрации, входа и выхода на базе встроенной и безопасной системы Django.

Каталог книг: Просмотр всех книг в библиотеке с возможностью динамического поиска по названию и автору.

Страницы книг: Детальная информация о каждой книге, включая обложку, описание и автора.

Интерактивная система отзывов: Авторизованные пользователи могут оставлять отзывы к книгам, которые сразу же отображаются на странице.

Загрузка изображений: Реализована загрузка обложек для книг через админ-панель.

Управление контентом: Простое добавление, редактирование и удаление книг и отзывов через стандартную админ-панель Django.

Технологический стек (пример):

Backend: Python 3, Django
База данных: SQLite 3
Frontend: HTML5, CSS3
Библиотеки: Pillow
